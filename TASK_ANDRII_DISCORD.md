## Завдання 1 — Створити бота (5 хвилин)

1. Відкрий https://discord.com/developers/applications
2. **"New Application"** → назва `cote-lapyx-bot` → **Create**
3. Лівe меню → **Bot** → **"Add Bot"** → **"Yes, do it!"**
4. Прокрути вниз → **Privileged Gateway Intents** → увімкни всі три:
   - ✅ Presence Intent
   - ✅ Server Members Intent
   - ✅ Message Content Intent
5. **Save Changes**
6. Натисни **"Reset Token"** → скопіюй токен
7. **Скинь токен Сергію приватно** (Telegram або ЛС)

> ⚠️ Токен нікуди не постиш — тільки приватно нам.

**Крок 8 — Запросити бота на сервер:**
1. Лівe меню → **OAuth2 → URL Generator**
2. Scope: ✅ `bot`
3. Bot Permissions: ✅ `Administrator`
4. Скопіюй посилання внизу → відкрий → вибери сервер **cote-lapyx** → **Authorize**
