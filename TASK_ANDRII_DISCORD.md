## Що потрібно зробити тобі

### 1. Створити Discord бота (10 хвилин)

Нам потрібен бот для сервера — він буде надсилати сповіщення, допомагати з командними задачами.
Ти реєструєш бота на свій Discord-акаунт.

**Крок 1 — Створити застосунок:**
1. Відкрий https://discord.com/developers/applications
2. Натисни **"New Application"**
3. Назва: `cote-lapyx-bot`
4. Натисни **Create**

**Крок 2 — Створити бота:**
1. В лівому меню → **Bot**
2. Натисни **"Add Bot"** → **"Yes, do it!"**
3. Ім'я бота: `CoteLapyx Bot`
4. Натисни **"Reset Token"** → підтвердь
5. **Скопіюй токен** (він показується один раз!)
6. Надішли токен Сергію приватно (Telegram / особисте повідомлення Discord)

> ⚠️ Токен — це пароль бота. Нікому більше не давай і не постиy в чати.

**Крок 3 — Налаштувати права бота:**
1. В лівому меню → **Bot**
2. Прокрути вниз до **"Privileged Gateway Intents"**
3. Увімкни всі три:
   - ✅ Presence Intent
   - ✅ Server Members Intent
   - ✅ Message Content Intent
4. Натисни **Save Changes**

**Крок 4 — Запросити бота на сервер:**
1. В лівому меню → **OAuth2 → URL Generator**
2. Scope: ✅ `bot`
3. Bot Permissions: ✅ `Administrator`
4. Скопіюй згенероване посилання внизу
5. Відкрий посилання в браузері → вибери сервер **cote-lapyx** → **Authorize**
